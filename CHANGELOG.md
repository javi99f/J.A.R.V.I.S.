# Historial de versiones

## 0.4.0

- Nuevo control del ordenador en Windows para abrir aplicaciones e interactuar con ellas mediante clics, escritura y atajos.
- Observación de pantalla bajo demanda para realizar acciones guiadas visualmente.
- Límites de seguridad que bloquean terminales, Registro, Ajustes de Windows, instaladores, rutas ejecutables arbitrarias y atajos críticos.
- Confirmación local para acciones sensibles e interruptor `Control del PC` en Ajustes.
- Holograma ámbar inspirado en la interfaz de JARVIS, con animación más visible y consumo ajustado.
- Tamaño mínimo de 170 px, visibilidad máxima duplicada y nodos orbitales siempre activos.
- Menú de clic derecho simplificado: solo muestra `Cerrar Jarvis`.
- Actualizador remoto para Raspberry Pi mediante GitHub Releases.
- Verificación obligatoria del tamaño y SHA-256, extracción segura, copia de seguridad y restauración automática.
- Conservación de `.env`, memoria, estado, configuración visual y dispositivos durante las actualizaciones.
- Botón `UPDATE` en la interfaz táctil y reinicio controlado sin cerrar la sesión gráfica.

Comprobado con 36 pruebas automatizadas.
